\section{2枚目 研究背景}
生き物の脳は多数の神経細胞によって構成されており、それらは電気的なネットワーク
を構成している. このネットワークの活動によって記憶、学習、身体制御などの脳の
機能は実現されていると考えられています. 

一方で単一の神経細胞についても、その形やコンダクタンスの分布のしかたが
神経細胞内の電位, 「膜電位」の変動に影響すると考えられています。

ここで用語の説明

膜電位とは, 細胞外を0としたときの細胞内の電位のこと。-70mVなら、外と比べて
細胞内が70mV低い電位であることを表す。

*図を使った生理学的な説明
神経細胞の内部や外部には様々な種類のイオンが存在する。
このイオンが細胞の内部に入ったり、出て行ったりすることで膜電位が変動する。

神経細胞の細胞膜上には特定のイオンのみを透過させる多数の種類のイオンチャネルが
存在する。

イオンチャネルには開いている状態と閉じている状態がある。
イオンチャネルが開いていないとイオンは透過することができない。

膜電位の挙動は微分方程式によって表現される

この式はA型Kチャネルというチャネルをモデリングしている。K+イオンを選択的に透過する
チャネルである。

n・l

\section{4枚目 目的}
先行研究では単一神経細胞の形態と機能の関係性の解明に対して
以下のようなアプローチを行った。

神経細胞の形態から、その機能を推定するのではなく、神経細胞の機能
から形態やコンダクタンス分布を推定する手法

問題点
1, 形態への評価は「長さ」が指標 <- 生命の生存競争の原理から考えると、細胞の形態
は小さい方がエネルギー消費が少なく、有利
しかし長さだけでは形態の評価は不適当
2, コンダクタンス分布、チャネルの分布も少ない方がエネルギー消費は少なく、
生存競争には有利
3, 線形分布では表現できない分布も存在する。

\section{5枚目 方法(機能)}
設定する機能input-order detection

2種類のシナプスを持つ神経細胞を想定する。
この神経細胞に対して2パターンの入力を与える。
1つ目は青いシナプス群をある時刻に活性化させ入力を与え、そのデルタtms後に
赤いシナプス群を活性化させるパターン。Blue -> Red
2つ目はその逆 Red -> Blue

図は2つの入力を与えた時の細胞体での膜電位変動。
青い線は1つ目の入力パターンを与えた場合、赤い線は2つ目の入力パターンを与えた場合
それぞれの膜電位の最大値をR{B->R}、R{R->B}とし、これを比にすると、
input-order detectionの機能がどれだけ実現されているかを定量的表す
ことができる。
